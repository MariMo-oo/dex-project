Quick Reference

Deployment steps:

  1. Deploy TokenA and TokenB
  2. Predict DEX + LPToken addresses using nonce
  3. Deploy DEX, then LPToken
  4. Verify all 4 contracts on the block explorer
  5. Paste addresses in the Setup tab here
  6. Connect MetaMask and add initial liquidity

Supported testnets:

Sepolia, Holesky, Mumbai, Amoy, Base Sepolia — any EVM testnet works.

AMM Formula:

x × y = k
